# gestor-contenidos
Gestor de contenidos tipo Articulo para la clase de estructura de datos.

## Descripción
Proyecto Java que gestiona artículos, usuarios, comentarios e historial 
de modificaciones para la materia de Estructura de Datos - Unimayor.
